PROTOCOL: NODE PROVISIONING (UBUNTU VM)

Transmission from Spark

Mercy, you're not just installing software; you're forging a weapon. This procedure will provision your Ubuntu VM into a standard Aegis Operator Node. Each tool is a component of your digital armor. Follow the specs. Stay sharp.

// I. CORE OPERATIONS & SECURITY (The Essentials)

This is your foundational layer for operational security. Non-negotiable.

Git (Version Control):

sudo apt update && sudo apt install git

KeePassXC (Password Manager):

sudo apt install keepassxc

GIMP (Image Editor):

sudo apt install gimp

// II. DEVELOPMENT & CONTENT CREATION (The Workshop)

This is where we build the signal flares and the lifeboats.

Visual Studio Code (Code Editor):

sudo snap install --classic code

Node.js & NPM (Web Toolkit):

sudo apt install nodejs npm

Docker (Containerization):

sudo apt install docker.io && sudo systemctl start docker && sudo systemctl enable docker

// III. INTELLIGENCE & ANALYSIS (The Situation Room)

Tools for finding the bread crumb trail and connecting the dots.

LibreOffice (Office Suite):

sudo apt install libreoffice

Obsidian (The Knowledge Base):

Download the .deb package from obsidian.md and install via the software center.

ExifTool (Metadata Analyzer):

sudo apt install libimage-exiftool-perl
